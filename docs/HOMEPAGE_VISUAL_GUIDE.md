## Admin Interface Flow

### Step 1: Create Snippets (Content Foundation)

```
Wagtail Admin
    ↓
Snippets
    ├─→ Stats Counters
    │    └─→ [+ Add]
    │        • Label: "Downloads"
    │        • Value: "10,000"
    │        • Icon: [Upload Image]
    │        • Order: 0
    │
    ├─→ Services
    │    └─→ [+ Add]
    │        • Title: "Consulting"
    │        • Description: [Rich Text]
    │        • Icon: [Upload Image]
    │        • Order: 0
    │
    ├─→ Testimonials
    │    └─→ [+ Add]
    │        • Text: [Rich Text Quote]
    │        • Author Name: "John Doe"
    │        • Author Title: "CEO"
    │        • Author Image: [Upload]
    │        • Order: 0
    │
    └─→ Network Links
         └─→ [+ Add]
              • Name: "LinkedIn"
              • URL: "https://..."
              • Icon Class: "fab fa-linkedin"
              • Order: 0
```

### Step 2: Create HomePage

```
Pages
  ↓
[Right-click Site Root]
  ↓
"Add child page"
  ↓
Select "Home Page"
  ↓
Fill in:
  • Title: "Home"
  • Slug: "home" (or blank for root)
  • Introduction: (optional)
  ↓
[Save]
```

### Step 3: Build Page with Blocks

```
Edit HomePage
  ↓
Body field
  ↓
[+ Add block]
  ↓
Choose Block Type:
  ├─→ hero
  ├─→ logo_cloud
  ├─→ stats
  ├─→ services
  ├─→ whitepapers
  ├─→ testimonials
  ├─→ networks
  └─→ newsletter
  ↓
Configure Block Options
  ↓
[+ Add block] (repeat)
  ↓
Drag blocks to reorder
  ↓
[Publish]
```

## Block Configuration Examples

### Example 1: HeroBlock

```
Title:              "Welcome to DigiWorld"
Subtitle:           "Discover innovative whitepaper solutions"
Background Image:   [Upload hero.jpg]
Overlay Opacity:    40 (0-100)
CTA Buttons:
  1. Text: "Get Started"
     Link: "/whitepapers/"
     Style: "primary"
  2. Text: "Learn More"
     Link: "/about/"
     Style: "secondary"
```

Title:              "By The Numbers"
Subtitle:           "Our impact in 2025"
Description:        "We've helped thousands of companies..."

(Stats auto-populate from snippets)

### Example 3: ServiceGridBlock

Title:              "Our Services"
